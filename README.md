# ツールの名前
Fit_pptx ツール

## 簡単な説明
UIにドロップした画像ファイルと等倍のpptxファイル（パワーポイント）を自動作成します。
px → inchの計算が面倒くさいので作りました。

## 必要要件
Windows11

## 使用ライブラリー
tkinterdnd2
PIL
python-pptx

## 使い方
1・Fit_pptx.pyを起動します。<br>
2・画像ファイルをUIの白枠にドロップします。<br>
3・Fit_pptx.pyと同一フォルダにoutput.pptxが作成されます。<br>

## インストール
解凍するだけ。アンインストールはフォルダごと消してください
